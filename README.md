# kana2kanji

[Google 日本語入力 - CGI API デベロッパーガイド](https://www.google.co.jp/ime/cgiapi.html)を使ったかな漢字変換プログラム

## usage

```js
import { kana2kanji } from "https://code4fukui.github.io/kana2kanji/kana2kanji.js";

console.log(await kana2kanji("さばえ"));
```
